URL Encode and Decode

Synopsys

URLs have special characters which have special meanings. For example, a ? means we're about to have some query parameters, an = is a parameter assignment, and a & separates parameters, among others. Whenever we stick interesting data in a URL, we must first encode it.

Strengths

This is a web standard for encoding data before adding it as a URL parameter.

Weaknesses

Encoded characters usually change from being one byte to three.
